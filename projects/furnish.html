Furnish

Furnish is a web design project for a fictional furniture and interior decoration agency. It is one of the side projects I worked on during my early days of learning frontend development.

The website has only three pages which are homepage, shop and account pages. This is not a real working ecommerce store because it there is no backend for it and neither is it connected to any server. All you see here is just the user interface.
