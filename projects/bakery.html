Bakery

Bakery is one of the side projects I embarked on to hone my front end development skill, as I've come to understand that one of the best way to improve one's coding skill is to build projects.

This is a multipages website built for a bakery and was built from scratch. The homepage which looks more like a landing page; the about page which contains info about the bakery and the chefs page which contains profiles of the chefs.

This is not a full stack web app, the booking sysytem is only done on the user inteface. Nothing is sent to the server.
